About Praksheet Repository for practice questions

Practice sheet application aims to assist educators , teachers and mentors to prepare practice sheets with ease. There are a lot of question choices available from a vast pool of contributed questions.

Use the input parameters on the "Sheets" page to search for specific criteria. Multiple options available such as Subjects Class Question Count Tags like add, subtract Language

You can also download the question set using the download button on top of the search result. Before downloading you can remove any unwanted question from the list. Do like a question in case its well written as that will assist others in using it

You need to login to submit a question, the more parameters you add to a query the more useful it will be for people to search for it.
